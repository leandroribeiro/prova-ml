## Tecnologias

### Backend

 - ASP .NET Core 3.1;
 - ASP .NET 5.0;
 - Swagger Tools;
 - XUnit (Testes de Integração);
 - Entity Framework Core (ORM);
 - DbUp (for Migrations);

### Frontend

- TypeScript
- Angular 8;

### Infra

- MSSQL Server 2019 Linux (Database); 
- Docker;
- Docker Compose;
- NGINX;
